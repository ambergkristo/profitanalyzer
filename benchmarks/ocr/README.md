# OCR Benchmark Fixtures

This folder contains deterministic, non-private OCR benchmark expectations.

- `expected/` contains synthetic expected JSON used by `npm run benchmark:ocr`.
- `private-samples/` is ignored and may hold local private invoice files for manual benchmark work.
- `private-results/` is ignored and may hold local private benchmark output.

Do not commit real supplier invoices, customer data, raw provider payloads, or private benchmark files.
